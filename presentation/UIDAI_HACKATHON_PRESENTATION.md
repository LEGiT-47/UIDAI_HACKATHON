## üé® DESIGN SYSTEM

### Color Palette
```
PRIMARY COLORS:
- UIDAI Blue: #004B87 (Headers, accent bars)
- Deep Blue: #002A4E (Dark text, emphasis)
- Light Blue: #E3F2FD (Background highlights)

SECONDARY COLORS:
- Orange: #FF6B35 (Alerts, stress indicators)
- Green: #2ECC71 (Success, stable indicators)
- Yellow: #F39C12 (Medium stress, warnings)
- Red: #E74C3C (High stress, critical items)

NEUTRALS:
- White: #FFFFFF (Backgrounds)
- Light Gray: #F5F5F5 (Alternate backgrounds)
- Dark Gray: #333333 (Body text)
- Medium Gray: #757575 (Secondary text)
```

### Typography
```
FONTS:
- Headers: Montserrat Bold, 44pt (Slide titles)
- Subheaders: Montserrat SemiBold, 32pt
- Body: Open Sans Regular, 18pt
- Captions: Open Sans Light, 14pt
- Numbers/Stats: Montserrat ExtraBold, 60pt

FALLBACK FONTS (if Montserrat/Open Sans unavailable):
- Headers: Arial Black
- Body: Calibri
```

### Layout Grid
```
MARGINS: 0.5 inch all sides
CONTENT AREA: 12.5" x 6" (within safe zone)
HEADER BAND: 1 inch height, UIDAI Blue background
FOOTER: 0.5 inch, light gray with page numbers
```
